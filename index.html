01 · IP 사용 조건

계약으로 정해진 것

계약 기간

  • 사실 최초 계약 3년 (2026년 10월 시작)
  • 사실 이견 없을 시 2년 연장
  • 사실 최대 5년 상표 사용 가능

앱 아이콘 내 IP 사용

  • 사실 골프예약 웹·앱 내부 사용 전제 — 앱 아이콘 캐릭터 사용 가능
  • 사실 앱 아이콘도 사용 범위·기간·최종 디자인 사전 감수 필요 (감수 = 상대의 사전 확인 절차)
  • 사실 아트웍 형태 제한 없음 (2D·3D 모두 가능), VX 희망 방향 반영
  • 사실 골프 라이언 5종 — 상호 합의 선정, 변형 없이 사용
  • 확인필요 알림톡·앱스토어 스크린샷 사용 미확정 — 별도 확인·제안 필요
02 · 배경과 의도

왜 다시 넣는가

사실 이 앱은 티타임 예약이 중심입니다. 결제와 예약 확인이 많습니다. 그래서 믿음이 중요합니다.

사실 앱 이름과 회사 이름이 같은 시기에 바뀝니다.

해석 이름이 한번에 바뀌면 이용자가 헷갈릴 수 있습니다. 서비스를 떠날 수도 있습니다.

제안 익숙한 라이언으로 "같은 서비스"임을 알립니다. 헷갈림을 줄입니다.

먼저 정할 것 — 새 이름에 '프렌즈'라는 말이 계속 남는지 먼저 정해야 합니다. 계속 남긴다면, 계약이 끝났을 때 대비가 더 중요해집니다.
03 · 아트웍 5종

포즈가 아니라 '역할'로 고른다

사실 사용할 5종은 카카오에서 실물 아트웍을 받아본 뒤 확정.

사실 단, 스윙하는 라이언은 확정 사용 — 앱 아이콘·스플래시에 적용.

제안 나머지는 쓰임새(역할)로 나눠 선정. 골프 동작 컷(스윙·클럽)이 골프 앱에 특히 적합.

04 · 배치 지도

어디에 · 어떤 컷을 · 왜

구분위치컷이유
필수스플래시스윙 라이언(확정)앱을 켰을 때 첫 화면. "같은 서비스"임을 먼저 알림.
필수앱 아이콘스윙 라이언(확정)가장 많이 봄. 그린 바탕. 노란색 안 씀.
필수웹 로그인②환영웹에서 처음 만나는 화면. 반기는 느낌을 줌.
필수웹→앱 브릿지①대표웹에서 앱으로 넘어가는 안내 화면. 브랜드를 각인.
필수이벤트 배너·페이지④골프동작눈에 잘 띔. 그린 바탕이라 노란색 규칙과 무관.
추천예약 성공③축하기분 좋은 순간. 캡처·공유가 자주 일어남.
추천환급캐시 안내 창③축하혜택을 받는 순간. 노란 코인을 라이언으로 바꿈.
추천빈 화면(검색결과 없음)②환영막다른 화면을 부드럽게. 다시 찾게 도움.
추천취소 확인 창⑤안내떠나려는 순간. 파란 3D를 라이언으로 바꿈.
협의필요앱스토어 스크린샷①대표설치 전 첫인상. 단, 사용 여부를 상대와 확인해야 함.
협의필요예약완료·초대 알림톡③축하 / ①대표공유·재방문에 좋음. 단, 사용 여부 확인 필요. 상단 노란 띠는 카카오 것이라 못 바꿈.
금지결제·약관·본인인증—믿음이 먼저. 캐릭터는 오히려 방해.
금지결제 실패 등 손해 알림—친근한 캐릭터가 분위기를 깸.
금지코스 상세·예약 달력—정보가 많은 화면. 깔끔하게 둠.
글자도 함께 바꿉니다. '카카오골프예약' → '프렌즈 필드예약'. '카카오 VX 통합멤버십' → 'VX 통합멤버십'. 챗봇 이름과 발신 프로필도 바꿉니다.
05 · 화면 예시

앱 · 웹 화면에 넣은 모습

웹 로그인 필수

06 · 로드맵

먼저 많이 쓰고, 뒤엔 줄인다

초기 (최초 3년)

라이언을 앞세운다

승인된 자리(스플래시·아이콘·웹 화면·예약 성공·이벤트)에 라이언을 넣습니다. "카카오골프예약이 프렌즈 필드예약으로 바뀐다"를 함께 알립니다.

연장 시점 (이후 최대 2년)

기대지 않을 준비를 한다

라이언 없이도 되는 아이콘·스플래시 대체안을 미리 만듭니다. 실제 성과를 보고 2년 연장을 정합니다.

프렌즈 필드예약 · 캐릭터 IP 적용 전략 · VX · 2026.07 — 화면 예시는 배치 위치를 보여주기 위한 것입니다. 원 표시 자리에는 승인된 라이언 아트웍을 넣어 씁니다. 글꼴(Pretendard)과 색은 실제 브랜드 기준에 맞춰 최종 정리합니다.
